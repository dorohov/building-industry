# ОрелСтройИндустрия

## Preloader
Страницы на которых подключается загрузчик
- Главная (index.html)
- Проекты (objects.html)
- Производство (production.html)
- Продукция (products.html)

Отключение/Включение - Подключение скрипта в **head**
